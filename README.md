# geektrust-problems

Node.js solutions and scaffolds for [Geektrust](https://www.geektrust.com) coding challenges.

## Layout

Each challenge lives in its **own folder** at the repo root. Use `run.sh` / `run.bat` inside a folder (or `npm start` per its README).

| Folder | Challenge |
|--------|-----------|
| [`metro-card/`](metro-card/) | [Metro Card](https://www.geektrust.com/candidates/coding/detailed/metro-card) |
| [`power-of-g-man/`](power-of-g-man/) | [Power of G-Man](https://www.geektrust.com/candidates/coding/detailed/power-of-g-man) |
| [`tame-of-thrones/`](tame-of-thrones/) | [Tame of Thrones](https://www.geektrust.com/candidates/coding/detailed/tame-of-thrones) (problem statement + `Problem2` messages) |
| [`geekdemy-node/`](geekdemy-node/) | Geekdemy Node scaffold |

## Conventions

- **Kebab-case folder names** (no spaces) so paths work the same on all shells and OSes.
- **One npm project per challenge** — `cd` into the folder before `npm ci` / `npm test`.
